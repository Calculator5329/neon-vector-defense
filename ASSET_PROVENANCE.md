# Asset Provenance

Neon Vector Defense separates source licensing from media asset rights.

## MIT-covered material

The repository's source code and documentation are licensed under the MIT
License unless a file has a more specific notice.

## Reserved media assets

The following directories contain AI-generated or project-specific media assets
and are not covered by the MIT code license:

- `public/art/`
- `public/audio/`

These assets were generated for Neon Vector Defense through local generation
scripts and AI media tooling, including OpenRouter-backed image/audio workflows.
They are included so the live game, local development build, recruiter demo,
and screenshots work out of the box.

Do not redistribute, relicense, sell, or reuse the media assets outside this
project without separate permission from the project owner.

## Committed demo/admin data

`public/balance-report.json` is intentionally tracked. It is generated by the
headless balance harness and gives the local admin dashboard useful demo data
without requiring private production telemetry.
